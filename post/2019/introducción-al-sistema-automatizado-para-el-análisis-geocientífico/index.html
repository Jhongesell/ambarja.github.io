¿Qué es SAGA?

.pull-left[ Es un GIS de código abierto que fue desarrollado en el lenguaje de C++ por del Departamento de Geografía Física de Göttingen y de Hamburgo en Alemania en los años 2001,a partir del año 2004 el código fuente fue mostrado al público y la comunidad científica bajo la licencia Open Source Software (OSS).

]

.pull-right[ SAGA inicialmente fue enfocado para el análisis digital del terreno,años posteriores esto ha ido evoluciando y hoy en día tiene diversos modulos para temas específicos como la: 1. Geoestadística 2. Teledetección 4. Caracterización hidrográfica 5. Climatología, entre otros.]

Linux

Si usas Linux va a depender del tipo de distribución que usas,para ubuntu 19.04 tu puedes instalar SAGA de la siguiente manera: En la terminal de comandos emplea el siguiente codigo:

sudo apt-get update
sudo apt-get install saga
